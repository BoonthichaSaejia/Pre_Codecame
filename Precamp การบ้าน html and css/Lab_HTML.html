2.Differentiation

Tag

Diffterence

hr & br

hr จะเว้นบรรทัดให้พร้อมเส้น ส่วน br จะเว้นบรรทัดเฉยๆ

ul & ol

ul ใช้กับลิสต์ที่ไม่ต้องเรียงเป็นลำดับส่วน ol ใช้กับลิสต์ี่เป็นลำดับ

select & checkbox & radio button

select ได้ผลออกมาคล้าย dropdown ส่วน checkbox เป็นกล่องให้ติ๊กถูกโดยไม่กำหนดจำนวนที่เลือกได้ แต่ radio button จะเลือกได้แค่ตัวเดียว แต่ลักษณะช่องคล้าย ๆ checkbox แต่เป็นวงกลม

div & span

div เป็นblock ส่วน spanจะติดกันเป็นบรรทัดเดียวยาวๆ

script & style

script เป็นคำสั่งของ javascript ส่วน style เป็นส่วนของ css
